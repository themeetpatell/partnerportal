Key Admin Pages

1. Overview Dashboard (Admin)
KPIs: Active Partners / Pending Approvals
KPIs: Open Leads / Conversion Rate
KPIs: Commissions Due / Overdue Invoices
Partner growth chart (monthly)
Lead funnel chart
Recent activity stream (all teams)
2. Partner Management
/partners — filterable table (type, status)
Pending tab → Approve / Reject / Request Docs
Partner detail: profile, docs, leads, commissions
Edit commission model
Suspend / reactivate partner
Manual channel partner creation
3. Lead Queue (Sales)
/leads — Kanban or table view
Columns: Submitted → In Review → Converted → Rejected
Assign lead to sales rep
Lead detail: customer info, docs, notes, timeline
Convert to Zoho Deal (one-click sync)
Add internal notes, request more info from partner
4. Service Queue (Ops)
/service-requests — filterable list
Assign to ops team member
Update status with notes
View / download documents from WorkDrive
Request additional docs from partner
Mark complete → triggers commission calculation
5. Commission Management (Finance)
/commissions
Pending commissions list (auto-calculated)
Bulk approve / reject
Payout queue: approve → trigger Stripe transfer
Commission model editor (flat % / tiered / milestone)
Dispute resolution workflow
6. Invoice Management (Finance)
/invoices
Auto-generate weekly/monthly invoices for channel partners
Review draft → Send to partner
Track payment status (Stripe webhook driven)
Overdue reminders (SendGrid)
PDF generation + tax invoice
7. Analytics (Admin)
/analytics
Partner performance table (leads, conversion %, earnings)
Commission reports by period / partner / model
Service delivery SLA tracking
Revenue attribution by partner
Export all reports to CSV / PDF
8. Settings (Admin)
/settings
Tenant branding (logo, colours, domain)
Commission model CRUD
Service catalog CRUD
Team member management + role assignment
Zoho / Stripe / SendGrid integration config
Notification templates editor
