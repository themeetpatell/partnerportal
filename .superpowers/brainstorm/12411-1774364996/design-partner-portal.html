Key Pages

1. Onboarding Flow (Pre-login)
/register → Step 1: Partner type selection
→ Step 2: Company & contact details
→ Step 3: Document upload (WorkDrive)
→ Step 4: Commission model selection
→ Step 5: T&C acceptance + submit
Referral: auto-approved → dashboard
Channel: pending review → email notification
2. Dashboard (Home)
KPI cards: Total Leads / Active Requests
KPI cards: Pending Commission / Paid
Recent activity feed
Quick actions (Submit Lead / New Request)
Notifications panel
Upcoming payment dates
3. Lead Submission (Referral)
/leads/new
Customer name, email, phone
Service interest (multi-select)
Notes / additional context
Document upload (optional)
→ Syncs to Zoho CRM as Lead
→ Status: Submitted
4. My Leads (Referral)
/leads
Table: name, service, status, date, commission
Status badges: Submitted | In Review | Converted | Rejected
Click → Lead detail with full timeline
Filter by status / date range
Export to CSV
5. Service Request (Channel)
/service-requests/new
Service catalog browser
Customer company & contact details
Service start date
Required document upload → WorkDrive
→ Status: Pending
6. Commission Dashboard
/commissions
Summary: Total earned / Pending / Paid
Table: source, amount, status, date
Commission model breakdown
Milestone progress tracker (if applicable)
Request payout button
7. Invoices (Channel)
/invoices
Invoice list: number, period, amount, status
Status: Draft | Sent | Paid | Overdue
Click → Invoice detail + PDF download
Pay via Stripe (embedded checkout)
Tax invoice download
8. API Access (Channel)
/api-access
Generate & manage API keys
Configure webhooks (URL + events)
API usage stats (calls / errors)
Link to API documentation
Webhook delivery logs
