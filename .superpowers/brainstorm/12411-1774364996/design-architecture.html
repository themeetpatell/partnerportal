Design Section 1 — System Architecture

Turborepo monorepo with two portals, shared packages, and Zoho ecosystem integrations

Monorepo Structure

Repository Layout
finanshels-partner-portal/
  apps/
    partner/                ← Partner-facing portal (Next.js 16)
    admin/                 ← Internal team portal (Next.js 16)
    api/                   ← Public REST API & webhooks (Next.js)
  packages/
    db/                    ← Drizzle ORM schema + Neon client
    types/                 ← Shared TypeScript types
    ui/                    ← shadcn/ui components (shared)
    commission-engine/     ← Commission calculation logic
    zoho/                  ← Zoho CRM + WorkDrive SDK wrappers
    notifications/         ← SendGrid email templates
    auth/                  ← Clerk helpers + RBAC middleware
